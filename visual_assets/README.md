# RoboCon Visual Assets

This folder contains presentation assets for RoboCon 2026.

## Included

- `robocon/robocon-theme.css` - Marp theme
- `robocon/robocon-bg.jpg` - Slide background extracted from official PPTX template
- `robocon/robocon-logo.png` - RoboCon logo extracted from official PPTX template
- `robocon/RBCN_2026_Online_Final.pptx` - Official source template
- `robocon/*.ttf` / `robocon/*.otf` - RoboCon fonts

## Usage

Marp is configured via `.marprc.yml` to use:

- `visual_assets/robocon/robocon-theme.css`

The instructor deck uses `theme: robocon` in frontmatter.

## Notes

Legacy presentation assets have been removed as part of the RoboCon cleanup.
